## Get Started

- `git clone https://github.com/maniadav/next-pwa-template.git`
- `cd next-pwa-template.git`
- `yarn install`
- `yarn dev`

**Also Checkout**

- [next-workbox-pwa](https://github.com/maniadav/next-workbox-pwa) for PWA using workbox-build package
